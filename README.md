# TP_PWEB - Web Programming Practical Work

A collection of web development projects demonstrating full-stack implementation across multiple technologies and architectural patterns.

## 📋 Projects Overview

### 1. **PediTiscosAPI**
A RESTful API for a food ordering system built with Java. This backend service handles order management, restaurant operations, and customer interactions.

**Technologies:**
- Java
- REST API architecture
- Backend service

**Features:**
- Order processing and management
- Restaurant data management
- API endpoints for food ordering operations

---

### 2. **GestãoLoja** (Store Management)
A comprehensive store/shop management system with web interface. Handles inventory, sales, and operational workflows for retail businesses.

**Technologies:**
- Java / Spring Framework
- Web interface
- Database integration

**Features:**
- Product and inventory management
- Sales tracking
- Store operations
- Administrative dashboard

---

### 3. **Frutilandia_Light_Moodle_F10**
A custom Moodle theme designed for the Frutilandia learning platform. Implements a light, modern interface for the Moodle learning management system.

**Technologies:**
- HTML/CSS
- Moodle theme development
- Frontend styling

**Features:**
- Custom Moodle theme
- Responsive design
- Light color scheme
- Educational platform integration

---
